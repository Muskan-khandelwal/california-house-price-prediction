# California House Price Prediction

## Overview
This project predicts California house prices using Machine Learning regression models.

The project compares:
- Linear Regression
- Polynomial Regression

to analyze prediction performance.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Features
- Data preprocessing
- Missing value handling
- Feature encoding
- Regression model training
- Model evaluation
- Visualization

---

## Dataset
California Housing Dataset from Kaggle.

---

## Results
The models were evaluated using:
- R2 Score
- RMSE

Polynomial Regression performed better on this dataset.
